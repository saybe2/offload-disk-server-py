Discord Webhook

Для работы хранилища необходимо указать Discord Webhook URL в файле .env

Как получить Webhook URL:
  1. Создайте Discord сервер или используйте существующий
  2. Перейдите в настройки канала → Интеграции → Webhooks
  3. Создайте новый Webhook и скопируйте URL
  4. Добавьте URL в DISCORD_WEBHOOK_URL в файле .env
